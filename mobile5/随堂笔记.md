
## 伸缩盒子中元素排序的属性
```html
    order: 值;

    特点：
        值越小，越靠前显示。
```

## 响应式布局
```html
    
    概念：同一个网页在不同设备中，实现不同的布局效果【rem适配可以适配，也可以不用做】
    
    原理（核心）：利用媒体查询的方式，实现不同设备中网页布局的差异化

    参考值：
    1. 大屏幕设备         大于等于1200px(min-width: 1200px)

    2. PC浏览器（默认）   大于等于980px(min-width: 980px)   

    3. 平板设备          大于等于768px(min-width: 768px) 

    4. 手机:            小于等于480px(min-width: 414px)
```
## Bootstrap 框架
```html

    作用: 实现快速响应式布局开发和移动项目开发。

    Bootstrap框架核心： 就是一个别人已经写好的一个CSS文件。

    学习的重点部分：

        1. 掌握（了解框架中提供的基础选择器）

        2. 栅格系统【就是用来实现相应式布局】


    步骤：
         1. 下载Bootstrap框架

            用于生产环境的 Bootstrap ： 下载下来的代码是经过压缩过后的

            Bootstrap 源码： 下载的代码是没有经过压缩的。

         2. 将下载好的框架（CSS文件）引入到html页面中

         3. 可以使用该框架中(CSS文件)中的选择器，栅格系统
```

## 开始学习基础的选择器和标签
```html
    
    1. 在框架中，用来表示标题的可以使用  .h1 - .h6 类名

    2. 表示小号文字标签  <small></small>

    3. 文字对齐： text-left   text-center  text-right 类名


    4. 将小写字母改变为大写字母：  text-transform: uppercase;

    5. 将大写字母改变为小写字母：  text-transform: lowercase;

    6. 将单词首字母转化为大写：  text-transform: capitalize;

    7. 将列表中的默认样式去掉： list-unstyled类名

    8. 实现li一行显示，给ul添加：  list-inline类名

```
## 框架中的栅格系统
```html

    Bootstrap框架中的栅格系统：

         重点： 
         
            1. 栅格系统最多可以将盒子分为 12列

            2. 栅格系统用到的布局方式（流式布局方式）

    使用：

        1. 使用栅格系统必须放到  container  或者 container-fluid 类名的盒子中

            container类名：表示当前盒子具有一个固定大小的宽度【也会随着浏览器视口的缩放，而改变】

            container-fluid类名： 表示当前盒子要将整个视口铺满【类似于手动设置宽度 100%】


        2. 使用栅格参数快速的实现相应式布局

             .col-xs-值：
                
                  xs ---> 适配手机设备的

             .col-sm-值:

                  sm ---> 适配平板设备

             .col-md-值

                  md ---> 适配默认浏览器

             .col-lg-值

                  lg ---> 超大屏幕的

            注意：
                 1. 在使用栅格参数的时候，如果栅格参数的总和大于12，那么多余的部分就会换行显示

                 2. 没有超过12的就会在一行显示
```
