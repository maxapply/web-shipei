# 移动web
```html
    核心： 将网页写到移动设备中 

    问题： 网页如何适配移动设备的屏幕大小

    目的： 通过技术解决 网页适配【可以让一个网页在不同设备中都能够正常预览】

    正常预览：

        1. 网页水平方向不能出现滚动条

        2. 网页在预览的过程中不能出现缩放效果

    掌握移动web的根本：

        能使用学习的知识点，解决页面适配效果
```
## 2D转化
```html

    ☞ 元素的位移： 改变页面中标签的位置

        元素：网页中的标签

        位移：改变元素的位置

        复习之前用来改变元素位置的方式：
             
             1. 浮动

             2. 定位（相对定位，绝对定位，固定定位）

             3. margin ， padding

        今天学习一种新的方式用来改变元素的位置： 2D转化中的位移。

    ☞ 元素的旋转:  让元素转圈


    ☞ 元素的缩放: 通过缩放改变元素大小。

         缩：缩小

         放：放大

    ☞ 元素的倾斜（了解）

         transform: skew(角度, 角度);

         总结：
             1. 第一个值代表让元素沿着水平方向倾斜

             2. 第二个值代表让元素沿着垂直方向倾斜

```
## 2D转化中位移
```html

    语法：
        transform: translate(水平方向X, 垂直方式y);

    总结：
        1. transform: translate(水平方向X, 垂直方式y); 一共有两个值设置

        2. 第一个值代表水平方向，第二个值代表垂直方向

        3. 可以只设置一个值，设置的这个值代表水平方向。

        4. transform: translate也可以设置负数，负数代表的就是相反的方向

        5. transform: translate可以设置百分比

        6. transform: translate可以设置百分比，相对元素自己的宽度（高度）

        7. 通过2D位移实现绝对定位的元素居中显示

              position: absolute;
              left: 50%;
              top: 50%;
              transform: translate(-50%, -50%);

        8. transform: translate 没有让元素脱标（人家只是在改变位置）
```

## 2D转化中旋转
```html
    语法：
        transform: rotate(30deg);
    注意：
        1. 旋转中的值单位是 deg

        2. 2D旋转默认是绕着当前元素的中心点旋转的。

        3. 通过 transform-origin 来改变2d旋转中心位置

            transform-origin: 具体的方位名称(left， top， right, center,bottom) 
            | 还可以设置具体值
            
        4. 如果设置的值是正数，那么代表是让元素顺时针旋转

        5. 如果设置的值是负数，那么代表是让元素逆时针旋转
```

## 2D转化中的缩放
```html

    语法：

        transform: scale(值)

    注意：
        1. transform: scale中的值，代表的是倍数

        2. 如果只设置一个值，那么代表当前元素的宽度和高度同时改变

        3. 如果设置两个值，第一个值代表宽度的倍数，第二值高度的倍数

        4. 如果设置的值大于1，那么代表元素放大对应倍数

        5. 如果设置的值大于0 小于1，那么代表元素缩小。

        6. 如果设置的值等于0，代表元素的宽度或者高度等于0

        7. 如果设置的值等于1，代表元素的宽度和高度不发生改变

        8. 不能设置负数。
```

## 3D转化
```html

    ☞ 3D转化的位移：本质也是用来改变元素的位置


    ☞ 3D转化的旋转：本质也是让元素转圈

          3d转化中的旋转，是让元素绕着轴（x, y，z）旋转。

    ☞ 3D转化的缩放：本质也是用来改变元素大小

    
    2D 和 3D的不同之处：

        2d 表示的是一个平面效果，2d平面中有两条轴 （x轴， y轴）

        3d 表示的是一个立体效果，3d立体有三条轴 （x轴，y轴，z轴）
```
## 3D转化中的坐标轴
```html

    x轴：水平从左向右

    y轴：垂直从上向下

    z轴: 从里向外的一条虚拟轴
     

    左手法则：
         大拇指向的方向就是  x轴的正方向
         食指指向的方向就是  y轴的正方向
         中指指向的方向就是  z轴的正方向
```

## 3D转化中的位移
```html

    语法：
        transform: translateX()  translateY()  translateZ();


    总结：

        1. 让元素沿着z轴位移的时候，浏览器中要出现一个近大远小的视觉效果

        2. 给位移元素的父元素设置一个透视属性【让浏览器渲染 近大远小的视觉效果】

            perspective: 1000px;

        3. perspective属性的取值可以是任意值，但是我们推荐使用 600-1000之间

        4. 位移中同样可以设置负数，负数代表沿着坐标轴相反的方向移动。
         
        5. 3D位移中，x轴，y轴可以设置百分比

        6. 百分比相对元素的宽度或高度

        7. 绝对定位的盒子居中显示，通过3d位移的方式实现：

              left: 50%;
              top: 50%;
              transform: translateX(-50%) translateY(-50%);
```

## 3D转化的旋转
```html

     语法：
        transform: rotateX(角度)  rotateY(角度)  rotateZ(角度);

     注意：
         1. rotateX 和 rotateY 是让元素绕着轴旋转

         2. rotateZ 本质上也是绕着轴旋转，但是我们视觉是看到的效果与 2d转化中的旋转效果是一样的。

         3. 当让元素绕着 X 轴 和 Y轴旋转的时候，要明确旋转方向（左手法则）

         4. 假如让元素沿着 X 轴旋转 60deg, 左手握着x轴，大拇指的方向和x轴的正方向相同，其他手指弯曲的方向就是旋转的方向

         5. 假如让元素沿着 X 轴旋转 -60deg, 左手握着x轴，大拇指的方向要和x轴的方向相反，其他手指弯曲的方向就是旋转的方向

         6. 如果要改变旋转轴的位置，可以通过 transform-origin 来设置
```
## 3D转化中的缩放
```html
    语法：
        transform:  scaleX(倍数)  scaleY(倍数)

    注意：
         1. 3d转化中的缩放的值也是表示倍数关系

         2. 如果要设置放大的效果，那么设置的值要大于1

         3. 如果要设置缩小的效果，那么设置的值大于0 小于1

         4. 如果要保持不变， 那么设置的值是1

         5. 如果设置的是0，宽度或高度为0
```
